dot-buddy

dot-buddy is an easy and efficient way to find ALL your midsets and more! Create a new dot sheet and transfer coordinates to calculate checkpoints from dot-to-dot, as well as see the step size and displacement vector for each move.

Add midsets like "6A" or "6B" simply by changing the # of the set. Swipe left or right to cycle through sets, and swipe up to return to the home screen.

Make your real-world dot book the best it can be with dot-buddy!

Documentation & How-To

The dot sheet list screen (EntriesController.swift). This screen houses all the dot sheets created by the user. Dot sheets are lists of coordinates which a performer marches between during the field show. Note: All dot sheets must have a unique name. To create a new sheet, tap the plus button in the upper right-hand corner. To edit an existing sheet, swipe right on its title. To delete a dot sheet, simply swipe left and hit the red delete button.

The coordinate editor (NewDotController.swift). This screen is responsible for displaying all of the coordinates contained in a given dot sheet. Each coordinate can be defined by a block of UI elements which are repeated across this screen. Every coordinate has a label, denoted by the "#" sign. By changing this text field to something like "6A," a subset will be created and the following coordinate labels will be adjusted. "S2S" refers to the side-to-side component of the coordinate, while "F2B" refers to the front-to-back component. The corresponding text fields for each component can only be left blank if "on" is selected. This is because if you are "on" a yardline or hash, your offset is 0 steps or nil.

The coordinate viewer (ViewController.swift). This screen displays each coordinate of the dot sheet. The default information, like ending position, set number, and counts is given at the top of the screen. The middle section displays all of the midsets for each count of the move. For example, an 8 count move will have 8 different checkpoints -- one for each step single-time. The bottom section displays the displacement vector for the total move, as well as the step size. "X.XX to-five" indicates the number of steps per 5 yards. "ZZ.Z st/count" denotes the number of 22.5" steps one must take each count to reach the final position.
